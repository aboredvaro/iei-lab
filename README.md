# Installation

First install nvm ([MacOS](https://github.com/nvm-sh/nvm#installing-and-updating))

Use node 14.16.0

``` bash
nvm install 14.16.0
nvm use 14.16.0
```

Use yarn

``` bash
npm i -g yarn
```

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:3001
yarn run dev

# build electron application for production
yarn run build

```
